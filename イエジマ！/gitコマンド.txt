git コマンド

cd :	 ChangeDirectory　現在いるディレクトリより下の階層のディレクトリに移動する
	今回は全員のリポジトリを"Ieshima"に統一したので、
	cd Desktop
	cd Ieshima
	と打ってから作業を開始する。

git add . :	データをリポジトリにプッシュする準備をするコマンド
		"."はすべてという意味
		とりあえずプッシュとプルの前に打つ

git commit -m "コメント"：	データをバックアップ領域に保存するコマンド
				これでバックアップファイルを生成する
				とりあえずプッシュとプルの前に打つ

git pull :	データをリポジトリからダウンロードするコマンド
		とりあえずプッシュの前に打つ

git push :	データをリポジトリにアップロードするコマンド
		データの変更があったらこまめにプッシュする
